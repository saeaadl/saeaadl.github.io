About SAE AADL

The SAE Architecture Analysis and Design Language — or AADL — is an SAE International standard for the modeling, analysis and generation of safety-critical system. It provides a precise modeling language to allow analysis of system designs (and system of systems) prior to development and supports a model-based, model-driven development approach throughout the system life cycle.
